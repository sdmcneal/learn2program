Add table

  3. Add a second row and make up values for each cell

Add form

Add a form to add new transactions and the functions to add. Read this first.

  1. Create a form with id=addform
  2. Add 4 inputs: tdate (text), ttransaction (text), ttype (radiobox with 2 options: deposit or withdrawal), tamount (text).
  3. Add a submit button within the form. Label it Add
  4. Create a javascript function that runs with the submit button is pressed. For now, just have the function create an alert box saying "Got it!" or something more clever. Name the function add_transaction(). Hint: add the click event to the submit button.

  6. Within the function, add a row with the values from the form. See here for an example. Don't worry about the balance column yet.
  7. Save the form by calling the function saveMyForm() and passing tdate, ttransaction, ttype, and tamount. See here for an example.
